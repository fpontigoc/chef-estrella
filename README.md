# Chef Estrella

Juego web simple para celular: se elige personaje, llegan clientes, piden platos, se eligen ingredientes, se cocina cuando corresponde y se entrega el pedido para ganar monedas y estrellas.

Personajes:

- Kevin Talks: parte con platos dificiles.
- Agustin: parte con platos moderados y luego dificiles.
- Isabela: parte desde platos basicos.

Desde nivel medio, si el pedido se prepara con ingredientes incorrectos, el cliente no paga y la cocina asume el costo de los ingredientes usados.

Para jugar, abrir `index.html` o usar la version publicada en GitHub Pages.
